أريدك أن تقوم بعملية استبدال كاملة لمزود الذكاء الاصطناعي في مشروعي، بحيث يتم إزالة Gemini نهائيًا من النظام، وتحديث المشروع ليعمل بالكامل باستخدام مزودات مجانية فقط، وهي:

1) Groq API

لتحويل الصوت لنص (Whisper) ولجميع مهام الـ LLM مثل تحليل النصوص والفورمات الذكي.

2) HuggingFace Inference API

لمهام الرؤية:

التعرف على الأنمي

التعرف على البودكاست من صورة الغلاف

التعرف على الشخصيات / الصور

أريد أن يتم:

أولًا: إزالة Gemini بالكامل

حذف كل الأكواد والدوال التي تستدعي Gemini.

حذف نظام تدوير مفاتيح Gemini.

تنظيف المشروع بحيث لا يوجد أي أثر لـ Gemini.

ثانيًا: دمج Groq API

أريدك أن:

تجعل كل التحويلات (صوت → نص) تعمل عبر Whisper الخاص بـ Groq حصريًا.

تجعل كل أدوات النصوص (تحليل، تلخيص، تنسيق ذكي، تعرف على البودكاست من النص) تعمل عبر موديل Llama من Groq.

إنشاء طبقة موحّدة للتعامل مع LLM بحيث يسهل استبدال المزود مستقبلًا.

استخدام متغيرات بيئة (ENV) لاختيار الموديل.

التعامل مع الأخطاء بطريقة واضحة للمستخدم.

ثالثًا: دمج HuggingFace Vision

إنشاء دالة موحّدة لكل أدوات الرؤية تعتمد على HuggingFace Inference API.

استخدامها في كل الأدوات: Anime / Podcast / Character.

المحافظة على نفس شكل النتائج للمستخدم.

رابعًا: Dashboard لمتابعة استخدام Groq والمزودات

أريد "لوحة تحكم AI" داخل لوحة الأدمن الحالية تشمل:

عدادات لعدد الطلبات اليومية/الأسبوعية.

نوع كل طلب (LLM / Whisper / Vision).

مزود الطلب (Groq / HuggingFace).

رسم بياني.

سجل آخر 50 عملية تمت.

عرض الأخطاء والتحذيرات.

نظام تنبيه لو حصل overload.

خامسًا: Smart Provider Routing

جهّز نظام يسمح مستقبلًا باستخدام أكثر من مزود ذكاء اصطناعي، بدون كسر الأدوات، عبر طبقة abstraction واحدة.

سادسًا: Page Rate Limiting

أريد تقييد عدد الطلبات لكل مستخدم بحيث:

يتم الحد من عدد تحويلات الفيديو/الصوت.

يتم الحد من استدعاءات LLM.

يتم منع الضغط الزائد على السيرفر.

عند تجاوز الحد: ترجع رسالة عربية واضحة.

سابعًا: Caching للطلبات

طبق كاشينج ذكي بحيث:

لو نفس الملف الصوتي تم رفعه قبل كده → رجع النص القديم بدون معالجة جديدة.

لو نفس صورة الأنمي أو البودكاست تكررت → رجّع النتيجة المخزنة.

لو نفس طلب النص تكرر → استخدم cache.

هذا يساعد في:
تقليل الاستهلاك + تسريع الأداء + تقليل الضغط على Groq.

ثامنًا: توضيح لي ما يجب عليّ إضافته في Render

بعد الانتهاء، أريد منك:

طباعة قائمة بكل متغيرات البيئة المطلوبة (ENV).

ما الذي يجب أن أضعه في Render؟

ماذا أحتاج من Groq؟

ماذا أحتاج من HuggingFace؟

وما القيم المطلوبة لكل متغير؟
